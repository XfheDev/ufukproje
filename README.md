# ChemLab Pro - Kimya Laboratuvarı Güvenlik Oyunu

Bu proje, öğrencilere laboratuvar ekipmanlarını ve kimyasal tehlikeleri interaktif bir şekilde öğretmek için tasarlanmış web tabanlı bir eğitim oyunudur.

## 🚀 Özellikler

- **Laboratuvardan Kaçış**: Gerçekçi senaryolar üzerinden doğru laboratuvar ekipmanını (Beherglas, Erlenmayer, Mezur vb.) seçme testi.
- **Tehlike Avcısı**: Laboratuvar tehlikelerini ve güvenlik önlemlerini (Korozif, Yanıcı, Toksik vb.) senaryo bazlı analiz etme.
- **Modern Tasarım**: "Neo-brutalist" ve "Glassmorphism" esintili, yüksek kaliteli ve şık kullanıcı arayüzü.
- **Puan Sistemi**: Süreye ve doğruluğa dayalı puanlama sistemi (Çerezler ile tarayıcıda saklanır).
- **Tam Türkçe Dil Desteği**.

## 🛠️ Teknolojiler

- **React**: Bileşen tabanlı mimari.
- **Vite**: Hızlı geliştirme ve derleme aracı.
- **Vanilla CSS**: Özel tasarım sistemi ve animasyonlar.
- **Lucide-React**: Yüksek kaliteli ikon seti.

## 📦 Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone <repo-url>
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı çalıştırın:
   ```bash
   npm run dev
   ```

## 🎨 Tasarım

Uygulama, göz yormayan pastel renk paleti (`#F9F3D3`, `#585C9A`, `#CBE3BD` vb.) ve keskin hatlara sahip modern bir arayüz ile geliştirilmiştir. Hiçbir gradyan kullanılmamış, bunun yerine katı renk blokları ve kalın bordürler tercih edilmiştir.
